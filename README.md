# RoseCraft üåπ

RoseCraft is a beautiful, interactive web application that allows you to create deeply personal rose-themed webpages for your loved ones. Perfect for Rose Day, anniversaries, or just to show someone you care.

Demo: [https://rosecraft.vercel.app/](https://rosecraft.vercel.app/)

## ‚ú® Features

- **Personalized Experience**: customize the recipient's name, your name, and your relationship.
- **Multiple Themes**: Choose from 5 stunning themes:
  - Classic Red
  - Soft Pink
  - Minimal
  - Dark Romance
  - Cute Cartoon
- **Interactive Rose Garden**: Choose the number of roses and assign a special meaning and message to each one.
- **Customizable Settings**:
  - Add background music
  - Secret Sender Mode (hide your name)
  - One-Time View option
- **Responsive Design**: Works beautifully on both desktop and mobile devices.
- **Animated Backgrounds**: Floating rose petals and dynamic gradients create a magical atmosphere.

## üöÄ How to Use

1.  Open `rosecraft.html` in your web browser.
2.  Click **"Create Your Rose Garden ‚ú®"** to start the builder.
3.  Follow the 5-step process:
    1.  **Basic Info**: Enter names and relationship.
    2.  **Theme Selection**: Pick a visual style.
    3.  **Rose Count**: Select how many roses to gift.
    4.  **Design Roses**: Click each rose to add a personal message.
    5.  **Final Touches**: Configure extra settings like music or secret mode.
4.  Preview your creation and share it with your special someone!
